# MeetFlow AI(회의록이 실행으로)
회의는 끝났지만, 실행은 자동으로 시작된다" AI 회의록 → 자동 실행 관리 SaaS

## 1. 프로젝트 개요
1.1 개발 목적
- 회의 후 수작업 정리 시간을 제거
- 실행 항목 누락 방지
- 조직 실행력 향상
- 자동 이메일 공유로 협업 비용 절감

1.2 핵심 기능 범위
- 음성 파일 or text 문서 입력
- 자동 전사
- 회의 요약 생성
- Action Item 자동 추출
- 지정 이메일 자동 발송
- 회의 요약 다이어그램 자동 생성

1.3고객의 문제점과 해결방안
회의는 했지만 실행 관리가 안 되는 회사, 학교 : 회의를 기록하는 것이 아니라 회의를 실행으로 전환하는 AI 도구로 문제를 해결한다.
